Title; Restaurant Recipe,
Created by; Antonio E Goncalves

Using Ruby, Javascript, HTML5, Bootstrap and more.

Purpose of app;

The idea is to create an app that allows the user to log in and find one self using google maps, search for restaurants in the vicinity 
and look at the recipes that the restaurant registered have in the menu.
Once the user finds the recipe it's looking for, try to cook it at home and find the chef inside.
